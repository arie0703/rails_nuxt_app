# rails_nuxt_app
Rails + Nuxt.jsによるアプリケーション  
フロントはNuxtで、apiをRailsで実装する。  

## 使用技術
### バックエンド
- Ruby 2.6.2
- Ruby on Rails 6.0.4
### フロントエンド
- Vue.js
- Vuetify
- Vuex
- Nuxt.js
- yarn 1.22.10
